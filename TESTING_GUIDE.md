# 🧪 خطوات الاختبار

## ✅ ما يجب اختباره

### 1. **تحميل الصفحة الأولى**

- [ ] عند فتح صفحة البيانات الشخصية، تظهر بيانات المستخدم الحالية
- [ ] الصورة الحالية تظهر في المربع العلوي
- [ ] اسم المستخدم والجوال يظهران كمعلومات قراءة فقط
- [ ] جميع الحقول تظهر فارغة (جاهزة للتعديل)

### 2. **اختيار الصورة**

- [ ] الضغط على الصورة يفتح معرض الصور
- [ ] اختيار صورة جديدة تحدثها على الواجهة فوراً
- [ ] الصورة تظهر بشكل صحيح مع حدود دائرية

### 3. **الملء والتحقق من البيانات**

- [ ] ملء حقل الاسم بقيمة صحيحة
- [ ] ملء حقل الجوال برقم صحيح
- [ ] ملء حقل المدينة
- [ ] ملء حقول كلمة المرور
- [ ] التحقق من الرسائل الخطأ عند ملء خاطئ

### 4. **حالات الخطأ عند الضغط على تعديل**

#### ❌ **بدون صورة**:

```
الرجاء اختيار صورة
```

#### ❌ **كلمات مرور غير متطابقة**:

```
كلمات المرور غير متطابقة
```

#### ❌ **بدون ملء الحقول المطلوبة**:

```
رسالة الخطأ من الـ validator
```

### 5. **النجاح**

- [ ] عند ملء جميع البيانات بشكل صحيح والضغط على "تعديل البيانات"
- [ ] يظهر مؤشر تحميل على الزر
- [ ] بعد الانتظار: رسالة نجاح باللغة العربية
- [ ] الرجوع التلقائي إلى الصفحة السابقة بعد ثانية

### 6. **معالجة الأخطاء**

- [ ] عند قطع الاتصال: رسالة خطأ واضحة
- [ ] عند رفع صورة كبيرة جداً: عرض خطأ
- [ ] عند فشل API: عرض رسالة الخطأ من الـ server
